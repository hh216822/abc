# 簡介
印尼設立海外投票機制，讓本國人在其他國家進行投票。
例如：印尼人來台灣工作，在印尼選舉時，可在台灣進行投票。
導致開票時間常常拉長至一個月或以上的時間，所以我們打算解決此項問題，
利用區塊鏈的不可修改性和公開性來進行此作業，來加速時程。
